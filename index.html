Gameplay

Le jeu se joue avec les flèches directionnelles du clavier :

  • Flèche gauche : barre vers la gauche
  • Flèche droite : barre vers la droite

Il est possible également d'y jouer avec la souris. Dans ce cas il suffit de diriger la barre avec la souris en faisant le mouvement approprié.

Une version mobile est disponible, il suffit d'appuyer sur l'écran à l'endroit voulu pour que se positionne la barre à cet endroit.

En appuyant sur la touche "p" du clavier (seulement sur ordinateur), il est possible de mettre en pause le jeu avant de revenir dessus.
